# UI/UX — vision “coworking” (signature V1)

## Intention
Les laboratoires (k0, k1, …) sont vécus comme des **espaces** de coworking :
- on “entre” dans un espace,
- on ouvre/ferme des zones,
- on collabore avec des “membres” (experts IA) qui font partie de l’équipe.

## Métaphore fonctionnelle
- Dashboard = hall / vue d’ensemble des projets et de l’activité.
- Chaque lab = une salle thématique (brainstorming, marché, branding, etc.).
- Les experts = membres d’équipe assignables à la salle.

## Layout direction (V1)
- Navigation latérale : organisations, projets, labs.
- Zone principale : “espace” courant (lab).
- Panneau contextuel : activité / décisions / tâches / score.
- Chat + composants dynamiques : intégré à l’espace (pas une page séparée).

## Signature minimale
- Sobriété premium, grandes marges, typographie lisible.
- Composants cohérents et “calmes” (éviter l’effet gadget).
- Accent visuel discret (couleur d’accent unique, non agressive).
