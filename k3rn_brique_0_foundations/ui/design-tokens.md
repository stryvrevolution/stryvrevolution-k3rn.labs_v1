# Design tokens (V1 — direction, non définitif)

Objectif : permettre une UI signature dès le départ sans bloquer sur un design final.

- Typographie : sans-serif moderne, lisibilité prioritaire.
- Radius : moyen (cohérence “premium”).
- Shadows : très légères.
- Couleurs :
  - Fond : clair, légèrement teinté
  - Surfaces : cartes avec séparation subtile
  - Accent : 1 couleur principale + 1 couleur de succès/alerte
- Iconographie : simple, consistante.

Note : les tokens exacts seront figés en brique UI dédiée, mais la cohérence doit exister dès V1.
