# Glossaire (V1)

- **Lab / laboratoire** : module fonctionnel (k0, k1, …) représentant un “espace” de travail. Plugin.
- **Espace** : représentation UI d’un lab (coworking). Peut être ouvert/fermé/navigué.
- **Expert** : profil IA fixe, spécialisé, connecté au Cortex.
- **User Kernel** : profil utilisateur transversal, stable.
- **Organisation** : tenant multi-projets, multi-membres.
- **Cortex** : graph + document vivant + vues (mind map).
- **Decision Ledger** : journal versionné, immuable, de tout ce qui a été décidé (ou proposé).
- **Intent Profile** : intentions humaines (résultats attendus) — projet-spécifique.
- **Compiled Blueprint** : sortie déterministe issue des intentions + contraintes.
- **Zip Master** : export structuré, lisible humain + exploitable IA.
