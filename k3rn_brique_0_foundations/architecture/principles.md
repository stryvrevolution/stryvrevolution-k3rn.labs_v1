# Principes non négociables

1. **Source de vérité** : le Cortex + Decision Ledger.
2. **Zéro dette structurelle** : pas de raccourcis qui cassent l’extensibilité.
3. **Déterminisme** : les recommandations sont compilées via règles versionnées.
4. **Traçabilité** : chaque recommandation/choix a un “pourquoi” explicite.
5. **Couches de données** : brut → interprété → structuré → validé.
6. **Labs plugins** : ajout facile de nouveaux labs, sans casser les précédents.
7. **Multi-tenant** : organisation dès V1 (membres, rôles, projets).
8. **Europe/RGPD** : hébergement et pratiques conformes (V1).
9. **UI signature** : expérience immersive “coworking” dès V1 (même minimale).
